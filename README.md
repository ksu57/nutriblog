# Заметки нутрициолога — локальная версия

## Быстрый старт

1. Распакуйте архив.
2. Установите зависимости:
   npm install
3. Запустите JSON-server (в отдельном терминале):
   npm run server
4. Запустите dev-сервер Vite:
   npm run dev
5. Откройте http://localhost:5173

## О проекте
- Главная страница: Обо мне
- Блог: статьи (импорт/экспорт/админ)
- Контакты: email и ссылки
- Дисконт NSP: лендинг с кнопкой регистрации

## Заметки
- Логотип и изображения включены в src/assets (если не видны, положите logo.png, verh.jpg, discont.jpg туда).
- Если нужно заменить изображения, положи новые файлы в src/assets и перезапусти dev-сервер.
